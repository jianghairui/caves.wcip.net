时逢九月，秋意渐浓

细碎的日光穿过枝叶映在斑驳的墙面上，小巷里只有一位环卫工人大叔簌簌地扫着落叶。我们转过几个弯，终于在尽头寻到了那处略显静谧的所在--杨柳青画社。

杨柳青画社始建于二零零八年九月，以收藏，研究，展示杨柳青木板年画为目的公益性博物馆。

博物馆客观地反映了杨柳青木版年画起源，繁荣，濒危，抢救，保护，传承和发展的历史过程。馆内珍藏自明代以来的杨柳青木板年画万余张，画版六千四百多块。其表现的内容包罗万象，有农耕生活，神话故事，节日习俗等，被称为民间大百科全书，对研究北方自明代以来的民风民俗和民间百姓的精神天地及文化连续性都具有重要作用。

杨柳青木板年画继承了雕版印刷技术，其刻工精美，绘制细腻，人物生动，色彩丰富而艳丽，表达了农耕社会的文明审美观，寄托了人们的希望和向往，成为了迎接除夕，纳福履吉的重要载体。

海河锦鲤，跃动津城——文创设计大赛

八月二十二日起，杨柳青画社为加快推进“五个现代化天津”建设，助力天津市申请联合国教科文组织“全球创意城市网络——设计之都”工作，坚定文化自信，挖掘天津优秀文化遗产，打造天津城市符号推出了以“海河锦鲤·跃动津城”为主题的，向全国征集“海河锦鲤”形象设计及文创产品设计的大赛。

这也正是我们走访杨柳青画社的目的。

本次杨柳青画社和山海文化公司合作，“海河锦鲤，跃动津城”设计大赛在山海文化旗下的山洞文创生态平台上开启了投稿通道。山洞平台近期上线，敬请关注投稿入口。

杨柳青画社的李部长告诉我们，虽然主题名字叫做“海河锦鲤”，但是希望大家能开拓思维，不要局限在鲤鱼的鱼形上。而是希望能将锦鲤进行变形，加入丰富的想象力和文化内涵，让锦鲤变成天津的一个独特的文化形象。让大家看到这个形象就能想到天津，感受到天津的亲切与魅力。同时将这一“文化锦鲤”应用于各类文创产品，逐步打造饱含地域特色和文化意识的“天津礼物”，使其成为有代表性的城市符号和文化品牌

鲤鱼不是鱼，希望各位有才华的设计师能从有形的鱼中看到无限的可能性。

山洞

山洞文创生态平台是为顺应国家文化产业发展需求，立足全国，助力各地文化创意产业的发展，为文创产品和制造企业的发展提供整体解决方案的文创生态服务体系。

山海文化公司凭借深厚的文化底蕴和过硬的技术实力以及丰富的业内经验，将文创产品从需求产生，到设计成稿，到生产落地，再到受用人群，打造了一个科学的生态圈。我们致力于以全新的服务体系，打造全国第一家文创生态孵化平台。

以“海河锦鲤”活动为例，杨柳青画社进行文创设计大赛活动发布，消费者提供创意，设计师上传投稿，创意排行榜，设计作品排行榜直观看到消费者喜好，入选作品由生产商接单，发起众筹，最后成品发往消费者。

山洞平台使文创产业链条的所有角色互联互通，特别是通过两个排行榜直观地为消费者需求提供了画像，能帮助博物馆美术馆，设计师，生产厂商更好的了解消费者。

未来全国将会有一千余家博物馆和美术馆图书馆等文化单位，百余家景区入驻并进行需求发布，全国范围内生产厂商入驻也在火热进行中。
